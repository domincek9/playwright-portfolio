# Playwright Automation Portfolio

## Project Overview

This project contains automated UI tests created with Playwright and TypeScript using the Page Object Model (POM) design pattern.

### Tested Application

SauceDemo
https://www.saucedemo.com

### Test Coverage

#### Login Tests

* Valid login
* Invalid password
* Missing username
* Missing password

#### Cart Tests

* Add backpack to cart
* Remove backpack from cart
* Add multiple products to cart

#### Sort Tests

* Sort products A-Z
* Sort products Z-A
* Sort products by price low to high
* Sort products by price high to low

#### Checkout Tests

* Successful checkout
* Checkout without first name
* Checkout without last name
* Checkout without postal code

#### Logout Tests

* Successful logout

* Playwright
